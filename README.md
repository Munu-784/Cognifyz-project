# Cognifyz Front-End Development Internship Project

## Project Overview
This repository contains the complete implementation of all 8 tasks for the Cognifyz Front-End Development Internship Program. The project demonstrates proficiency in HTML, CSS, JavaScript, responsive design, API integration, form validation, Bootstrap framework, and Sass preprocessing.

## Tasks Completed
1. **Level 1: Beginner**
   - Task 1: Basic HTML Page Structure
   - Task 2: Styling with Inline CSS

2. **Level 2: Intermediate**
   - Task 3: Responsive Design with Media Queries
   - Task 4: Interactive Button with JavaScript

3. **Level 3: Advanced**
   - Task 5: API Integration (JSONPlaceholder)
   - Task 6: Form Styling and Validation

4. **Level 4: Expert**
   - Task 7: Component-Based Styling with Bootstrap
   - Task 8: CSS Preprocessing with Sass

## Features
- Fully responsive design that works on all devices
- Interactive theme toggle button
- Dynamic data fetching from JSONPlaceholder API
- Client-side form validation with real-time feedback
- Modern UI with Bootstrap components
- Sass preprocessing with variables, mixins, and nested styles
- Clean, well-commented code

## How to Run
1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. No additional setup required (all resources are CDN-based)
